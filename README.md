# 📖 Project Overview

UOMO is a modern, stylish, and fully responsive e-commerce web application designed to deliver a smooth and engaging online shopping experience. The project focuses on clean UI/UX design, dynamic product browsing, and modern frontend development practices using React technologies.

The website allows users to explore collections, view product details, manage shopping carts, and interact with a visually appealing shopping interface optimized for all devices.

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/faisalbari193/Ecommerce-Project.git
```

---

## Navigate to Project Folder

```bash
cd Ecommerce-Project
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```
